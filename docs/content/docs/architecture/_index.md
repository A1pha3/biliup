## 章节概览

本章节包含以下内容：

### 架构概览
了解 biliup 的整体架构设计，包括前端层、后端层和引擎层的职责划分，以及各组件之间的通信方式。

### 前端架构
深入了解基于 Next.js 的前端实现，包括技术栈选型、项目结构、API 交互和 WebSocket 实时通信。

### 后端架构
探索 Rust 后端服务的设计，包括 Web API 服务器、核心上传库、数据库设计和认证机制。

### Python 引擎
理解 Python 引擎的工作原理，包括下载器实现、上传器实现、任务调度和并发控制。

### 数据流设计
通过数据流图了解录制和上传的完整流程，以及任务状态更新和日志传递机制。

### 插件系统
学习插件系统的设计原理，包括插件基类、装饰器模式、注册机制和事件驱动框架。

### 设计决策
了解 biliup 的关键技术决策和权衡，包括为什么选择混合架构、上传线路选择策略等。

## 适用读者

- **高级用户**: 希望深入了解系统工作原理，进行高级配置和优化
- **开发者**: 准备参与项目开发或基于 biliup 进行二次开发
- **架构师**: 研究混合架构设计和技术选型
